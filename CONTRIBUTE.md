# Contribute

This document outlines how people can contribute to this project, if contribution is permitted at all. People are divided into the following categories: project owner(s), team members, general public. The project owner(s) can do as they like. Team members, and the general public, if allowed to contribute, can only do so as defined in this guide.

## Team Members

Team members are official developers for the project. If contribution by team members is applicable to this project, and permitted, their contributions must adhere to the stated requirements, and be done in the stated manner. Even if done correctly, this does not mean the contribution will be accepted into the project.

- Status: Not applicable.
- Requirements:
 - Contributions must be clearly dedicated to the Public Domain using the Ritchey Public Domain Dedication v1, or the CC0 1.0 Universal dedications. This allows the project to freely use the contributions, including re-licensing them under whatever license(s) this project uses.
